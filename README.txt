Expiry Date Tracker:

Description:

This project is designed to help you keep track of the expiration dates of your food items. By entering the product name and expiry date of each item, you can keep track of them before they go bad.

Features:

Add food items: Easily add new food items to your inventory.
Edit food items: Edit the already set product name and expiry date.
Delete food items: Remove already expired or unwanted items from the list.
Set expiration dates: Specify the expiry date for each item.
